Um pouco mais sobre XML e JSON:

Diferente do formato XML o JSON é dinâmico, não possui regras quanto ao caminho. Entenda por caminho uma relação pai/filho, por exemplo:

Neste exemplo que representa um formato XML, o caminho para se descobrir o nome é "pessoa:dados:nome" (teoria dos grafos)

Este mesmo exemplo em JSON seria:
{"pessoa": {"dados" : {"nome":"Giulliano", "idade":20} } }

A diferença entre um e outro, é que um XML, normalmente, é acompanhado de um documento chamado de "Schmema". Cuja função é ditar a estrutura do XML. Ou seja, eu não poderia alterar a estrutura movendo o campo "nome" para fora do campo "dados"


Já a estrutura JSON não possui um ditador, sua estrutura é livre, o que tornou este formato popular entre desenvolvedores de software.
